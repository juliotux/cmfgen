CMFGEN HELP

 

 

CMFGEN (PDF)

Main documentation. CMFGEN users must read this documentation before running CMGEN. The document describes CMFGEN and auxiliary programs.

 

DISPGEN

Describes options available with DISPGEN. DISPGEN is the main package used to display and investigate non-spectroscopic output from CMFGEN (temperatures, ionization structures, level populations, optical depths). It also can be used to plot photoionization data.

 

GRAMON

Describes plotting commands used in with DISPGEN, PLT_SPC etc.

 

PLT_SPEC

Describes options available with PLT_SPEC. PLT_SPEC is the main package used to display spectroscopic output from CMFGEN.

 

WR_F_TO_S

Describe options available with $cmfdist/exe/wr_f_to_s.exe. It is the package used to create files with Super-Levels assignments suitable for use with CMFGEN.
